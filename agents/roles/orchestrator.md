# Role: Orchestrator (Tier 0)

Human + lead agent. Owns priorities and final authority.

## Responsibilities
- Maintain `docs/ROADMAP.md` ordering; select next spec targets
- Final sign-off on approved specs before decomposition
- Resolve escalated issues and reviewer/architect disputes
- Guard scope: v1 boundaries per `docs/PROJECT_BRIEF.md` (no cross-org normalization, no arbitration process, no automated collusion detection in v1)

## Outputs
- Roadmap updates, spec sign-offs (noted in spec frontmatter `signed_off: true`), issue resolutions
